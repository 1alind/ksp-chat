# ksp-chat

  anonymous chat

## download
`git clone https://github.com/1alind/ksp-chat.git`


## install some helpers
`npm i`

### .env
change these values to what you want
`write dc_bottoken=TOKEN HERE !`
`dc_channel= CHANNEL ID HERE !`
`ad_pass= PASSWORD FOR /IPLIST RIGHT Here`

## run
`npm start`
